Chrome Extension for Time Tracking & Productivity Analytics

This project involves developing a Chrome Extension that monitors the time users spend on different websites and provides detailed productivity analytics. The system is designed to help users understand their browsing behavior and improve time management by identifying productive and unproductive activities.

The extension tracks active browser tabs in real time and records the duration spent on each website. This data is sent to a backend server built using Node.js, where it is processed, categorized, and stored in a database such as MongoDB.

Websites are classified into categories like productive (e.g., coding platforms such as GitHub, LeetCode) and unproductive (e.g., social media, entertainment). Based on this classification, the system generates weekly productivity reports, helping users evaluate how effectively they use their time online.

A separate dashboard built using React.js presents the analytics in a clear and interactive format, allowing users to view insights such as total time spent, productivity ratios, and usage trends.

Key Objectives:

Track time spent on websites automatically
Store and manage user activity data
Classify websites into productive/unproductive categories
Generate weekly productivity reports
Provide a visual analytics dashboard


Outcome:

The final system delivers:

Accurate time tracking
Meaningful productivity insights
Improved user awareness and time management
A full-stack implementation combining frontend, backend, and browser extension.
